What's inside

Built to update once, use everywhere

Whether you tweak typography, swap colors, or add a new helper function, the change propagates across all tools automatically.

Tailwind + base.css

Tailwind powers rapid layout while `assets/css/base.css` handles tokens, focus states, and shared micro-interactions. Tools stay consistent even as you expand the kit.

:root {
  --tt-color-primary: #2563eb;
  --tt-radius-lg: 1.25rem;
}

Modular JavaScript

Shared helpers live in `assets/js/site.js`, while each tool ships with its own lean module. No frameworks required, just native web APIs.

TeacherToolkit.storage.write("timerDuration", seconds);
